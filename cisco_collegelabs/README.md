# Cisco College Labs

This repository contains my completed Computer Network and Security course from college. 

## Contents

- **Network Topologies** – Diagrams and designs of lab networks.  
- **IP Addressing** – Subnetting and addressing plans used in labs.  
- **Cisco IOS Configurations** – Router and switch configurations from exercises.  
- **Routing Protocols** – Labs featuring DHCP and static routing.  
- **Troubleshooting Notes** – Problems encountered in labs and how they were solved.  

## Purpose

The repository serves as a record of my practical skills in networking, showing my ability to:

- Configure and verify routers and switches  
- Implement and troubleshoot routing protocols  
- Design and document network topologies  
- Solve common networking issues  

## Tools Used

- **Cisco Packet Tracer** – For lab simulations and exercises.  
- **Cisco IOS** – Commands and configurations applied on devices.  
- **Documentation** – Notes, diagrams, and screenshots from labs.  

## Usage

- Browse lab folders to review exercises and configurations.
