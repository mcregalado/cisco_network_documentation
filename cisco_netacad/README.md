# Cisco Packet Tracer Exercises

This repository contains my completed exercises from Cisco Packet Tracer courses, demonstrating hands-on skills in networking and IoT simulations. It includes network topologies, configurations, and notes from my lab work.  

## Completed Courses & Exercises

1. **Getting Started with Cisco Packet Tracer**  
   - Basic network setups  
   - Router and switch configurations  
   - IP addressing and subnetting exercises  
   - Connectivity verification and troubleshooting  

2. **Exploring IoT with Packet Tracer**  
   - Connecting IoT devices in simulated networks  
   - Configuring smart sensors and actuators  
   - Implementing IoT protocols  
   - Testing IoT scenarios in Packet Tracer  

3. **Exploring Networking with Packet Tracer**  
   - Advanced routing and switching exercises  
   - VLANs, static and dynamic routing  
   - Troubleshooting complex network topologies  
   - Verifying connectivity and network performance  

## Features

- Step-by-step lab exercises I completed

- Network topologies I designed  
- Troubleshooting notes and solutions from my practice

This repository serves as a portfolio of my practical skills in Cisco networking and IoT, demonstrating my ability to design, configure, and verify networks using Packet Tracer.
